# Hippie Completion
Hippie Completion plugin for VSCode

Hippie Completion a.k.a cyclic word expand in JetBrains, or simple tab-completion in some editors.

### Setup key binding

```json
    {
        "key": "tab",
        "command": "hippie-completion.next",
    },
    {
        "key": "shift+tab",
        "command": "hippie-completion.prev",
    },
```
